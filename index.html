WebServer

Resumen

Arduino Ethernet Shield le permite conectar fácilmente su Arduino a Internet. Este escudo le permite a su Arduino enviar y recibir datos desde cualquier parte del mundo con una conexión a Internet. Puede usarlo para hacer cosas divertidas, como controlar robots de forma remota desde un sitio web, o hacer sonar las campanas cada vez que recibe un nuevo mensaje de Twitter. Este escudo abre un sinfín de posibilidades al permitirle conectar su proyecto a Internet en un momento dado.
